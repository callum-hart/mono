# Example UI - Facebook

**Credit:** [Ben Hartley](https://dribbble.com/hartleydesign) for his awesome design on [dribbble](https://dribbble.com/shots/1666016-Facebook-redesign).

**To run:**

- install Sass
- `npm install`
- `python -m SimpleHTTPServer`
- `grunt watch`
- open `localhost:8000/facebook.html`
